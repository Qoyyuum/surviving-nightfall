# Surviving Nightfall

A 3D third-person bullet hell survival game built with Python and Ursina Game Engine.

## Quick Start

```bash
# Install dependencies
uv sync

# Run the game
uv run python main.py
```

## Features

- 🎮 Third-person shooter controls (WASD + mouse)
- 🌊 5 wave progression system
- 👾 5 unique enemy types with special abilities
- 🔫 7 unlockable weapons
- ⚡ 3 special abilities with upgrades
- 💾 Persistent score and weapon unlocks
- 🎯 Roguelike permadeath gameplay

## Game Overview

Survive waves of supernatural enemies in an endless night. Fight through 5 increasingly difficult waves using various weapons and special abilities. Unlock new weapons with accumulated score across sessions.

## Requirements

- Python 3.13+
- Ursina Game Engine 8.3.0+
